# Deriving Density Fields

A practical method for deriving a coherent domain as a Density-field and comparing domains without collapsing their internal structure.

## What this is
This is a practical manual for:
- deriving a domain’s field
- judging whether the derivation is real
- comparing domains correctly

## Start Here
Read Sections 1–5 for the core method, then use the Field Manual appendix for practical application.
